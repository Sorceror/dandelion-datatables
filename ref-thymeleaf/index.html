Thymeleaf reference

Thymeleaf is a recent and very promising template engine that can completely substitute JSP in your applications.

Since v0.8.1, you can use Dandelion-DataTables with Thymeleaf thanks to the Dandelion-DataTables dialect. At the moment, not all options are supported but more will be added over the time !

You can find below the dialect reference, sorted by HTML tags :

  • <table> attributes
  • <thead> attributes
  • <th> attributes
