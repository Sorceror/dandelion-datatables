Column tag


Description

Describes a column of the HTML table.


Usage

You can either let the property attribute handle the content of a cell. This attribute is used to retrieve the corresponding value of the bean of the iterated collection. As a consequence, the column tag doesn't need to have a body. 

<datatables:column title="..." property="..." />

Or you can set a body and put anything you want inside (HTML, other tags...). See the row table attribute which is useful in that case.

<datatables:column title="...">
    <%-- Anything you want here ! --%>
</datatables>
